# Docker

## Installation by docker compose

```bash
$ docker-compose up -d --build client-angular-ui
```

## Considerations
For docker integration we need to change the DATABASE_HOST variable in .env file
from DATABASE_HOST='localhost' to DATABASE_HOST='mysqldb-app'

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# test coverage
$ npm run test:cov
```
